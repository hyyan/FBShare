Integrating Testing Platform With Facebook Share functionality

The idea of this demo is to answer simple math question , if you answerd the question correctly , you will get a message says that you have answered correctly and a button which will allow you to share your result on facebook , if not a simple "wrong question" message will be shown with "Try Again" button. We use the default facebook share api to controll the following elments which will be shared

  • Title : the post title
  • Description : the post description
  • Picture : the post picture , for now it is static image to create dynamic images we need a service like http://dummyimage.com/
  • href : the post link , it could be the website url
